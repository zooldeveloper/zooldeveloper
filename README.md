# 💫 About Me:
## Hi there 👋<br>
I'm Mukhtar - aka [Zool Developer](https://www.zooldeveloper.com) 😎<br><br>
I started teaching myself web development in early 2021 during the COVID-19 pandemic and turned that curiosity into a practical full-stack skillset. Today I build modern fullstack applications with a strong eye for usability and visual clarity, primarily using Vue, Nuxt, Node.js, and PostgreSQL. Across client platforms, business tools, and product experiences, my goal is to create something useful, easy to navigate, and solid enough to keep evolving.
